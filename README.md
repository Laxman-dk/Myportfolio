# Laxman Kadasidda — SOC Analyst Portfolio

🔗 **Live Portfolio:** [https://laxman-dk.github.io/Myportfolio](https://laxman-dk.github.io/Myportfolio)

A cybersecurity-themed personal portfolio with full 3D animations, matrix rain background, neon effects, and responsive design.

## Features
- Matrix rain background animation
- 3D tilt cards with mouse parallax
- Neon cursor trail
- Typed text cycling through roles
- Scroll reveal animations
- Animated stat counters
- Ripple button effects
- Particle burst on click
- Floating ambient particles
- 3D flip skill cards
- Fully responsive

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts: Orbitron, Rajdhani, Share Tech Mono
- Font Awesome 6.5 Icons

## Sections
1. Hero — Photo, name glitch, typed role, social links
2. About — Bio, quick info card, animated stats
3. Skills — 3D flip cards (hover to reveal tools)
4. Certifications — CompTIA, Google, Cisco, THM/HTB
5. Projects — AI Workforce Risk System + SOC Home Lab
6. Experience — AICTE SOC Internship + Education timeline
7. Contact — Contact info + message form
